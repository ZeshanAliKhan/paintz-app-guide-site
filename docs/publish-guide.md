# Publish Guide

## Expected repository name

`paintz-app-guide-site`

## Expected GitHub Pages URL

`https://zeshanalikhan.github.io/paintz-app-guide-site/`

## Steps

1. Create a public GitHub repository named `paintz-app-guide-site`.
2. Push this project folder to the `main` branch.
3. Open repository `Settings`.
4. Open `Pages`.
5. Set:
   - `Deploy from a branch`
   - `main`
   - `/root`
6. Save.
7. Wait for the live Pages URL to return `200`.
