# PaintZ App Guide

PaintZ App Guide is a separate GitHub Pages-ready static site project built around the keyword `paintz app`.

This project is intentionally separate from earlier GitHub Pages projects in this workspace. It uses its own folder, files, page structure, copy plan, and publish path.

## Project goal

Create a small, useful site for people searching for PaintZ app information, especially users who want a lightweight browser drawing tool, Chromebook drawing app, or MS Paint-style editor.

## Planned structure

- Home / PaintZ App Guide
- My Practical Notes on PaintZ App
- PaintZ for Chromebook
- PaintZ Features and Limits
- PaintZ App FAQ
